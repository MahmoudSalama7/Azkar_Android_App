#Azkar
Azkar is a mobile application that provides a variety of useful features for Muslims, including azkar (remembrance of God), tasbeeh (prayer beads), prayer times, and explanations of Quranic verses. The app is built using Java and utilizes an SQLite database for user sign-in and registration.

Features
Slider view with azkar
Tasbeeh tracker
Prayer times and notifications
Quranic explanations
Sunnah reminders
Requirements
To run this app, you will need the following:

A device running Android 4.4 or higher
An internet connection for some features (such as prayer times)
How to Use
Download and install the app from the Google Play Store.
Sign up for an account or log in with an existing one.
Use the app's features as desired.
